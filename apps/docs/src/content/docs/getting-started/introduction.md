# Minions Alerts

Alert rules, delivery preferences, and alert history for availability changes

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-alerts/sdk`) — Core library for Node.js
- **Python SDK** (`minions-alerts`) — Python bindings
- **CLI Tool** (`@minions-alerts/cli`) — Command-line interface
- **Web App** — Interactive playground at [alerts.minions.wtf](https://alerts.minions.wtf)
