---
title: Installation
description: Install Minions Alerts packages
---

## TypeScript / Node.js

```bash
npm install @minions-alerts/sdk minions-sdk
```

## Python

```bash
pip install minions-alerts
```

## CLI

```bash
npm install -g @minions-alerts/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
